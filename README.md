
# Name

fl7.brailleArt - Make the Braille Pattern art from a matrix of boolean.

# Synopsys

```
$ fl7 'USE(MODULE(FILE; "fl7.brailleArt")).encode([y : 0 .. 100 | [x : 0 .. 100 | ((x - 50) ^ 2 + (y - 50) ^ 2) <= 50 ^ 2]])'
```

# Installing

## 1. Install fluorite-7

https://github.com/MirrgieRiana/fluorite-7

## 2. Install this module

```
$ fl7m pull 'https://raw.githubusercontent.com/MirrgieRiana/fl7-module_fl7.brailleArt/main/module.json?'
```

# Usage

## `_.encode(booleanTable)` function

- `booleanTable`: bitmap image as an array of array(s) of boolean
- Returns: braille pattern art as a multiline string
